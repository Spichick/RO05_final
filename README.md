# RO05 Final Project

## Description

Ce dépôt contient l’ensemble des travaux pratiques (TD) réalisés dans le cadre du cours RO05.

Le projet est organisé en **4 TD distincts** :

- TD1  
- TD2  
- TD3  
- TD4  

## Organisation du dépôt

Chaque dossier de TD contient :

- un fichier `README.md`
- plusieurs fichiers de code associés

## Contenu des README de chaque TD

Dans chaque `README.md`, vous trouverez :

- une **description de la structure du code**
- les **instructions détaillées pour exécuter les programmes**
- des **réponses aux questions posées dans le TD**, lorsque nécessaire

Ces fichiers permettent de comprendre rapidement le fonctionnement de chaque TD sans avoir à analyser directement tout le code.
